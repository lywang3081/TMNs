# [Triple Memory Networks: a Brain-Inspired Method for Continual Learning (TNNLS 2021)]() 


## Execution command
To run the cifar-10 experiment:
```
cd cifar10
python run_TMNs_cifar10.py
```

To run the ImageNet-50 experiment:
```
cd imagenet-50
python run_TMNs_imagenet.py
```
